# Retail Performance & Returns Analysis (Tableau)

## Project Overview
This project analyzes retail sales and return data to evaluate overall business performance, profitability by category and region, and the financial impact of product returns.

## Tableau Dashboard
View the interactive dashboard here:  
[Tableau Public Link](https://public.tableau.com/app/profile/blessing4552/viz/RetailPerformance_17696430803960/Dashboard1?publish=yes)

An interactive Tableau dashboard was built to help stakeholders monitor KPIs and identify areas for improvement.

## Key Features
- KPI tiles: Sales, Profit, Average Order Value (AOV)
- Parameter-driven metric selector
- Regional and category performance analysis
- Returned vs non-returned order comparison
- Interactive filters and action-based highlighting
- Tableau Story for executive-level insights

## Data Sources
- Orders: transactional sales data
- Returns: returned order information
- People: regional management lookup




## Business Insights
- Technology category drives the highest profitability
- Furniture shows high sales but lower margins
- Returned orders significantly reduce profit across categories
- Certain regions consistently outperform others
